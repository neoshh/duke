# User Guide

## Features 

### Add task 
    
### Delete task
    
### Mark task as done
    
### Display tasks
    
### Search for task


## Usage

### `Keyword` - Describe action

To add task, type:
    `todo` + 'name of task'
    `deadline` + 'name of task' + 'DD/MM/YY hhmm'
    `event` + 'name of event' + 'DD/MM/YY hhmm'

To delete task, type:
    `delete` + 'task number'
    
To mark task as done, type:
    `done` + 'task number

To display all tasks, type:
    `list`
    
To search for tasks, type:
    `find` + 'keyword'
